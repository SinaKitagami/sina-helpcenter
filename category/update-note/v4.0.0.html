Version 4.0.0 Update Note

Note

App Build: ad31a0b

サーバリプレイス

  • アプリ実行環境を新サーバへ移行しました。
  • データベースサーバを新サーバへ移行しました。
  • サービス監視サーバを新サーバへ移行しました。
  • アプリサーバ・データベースサーバ・サービス監視サーバのOSを更新しました。
    Ubuntu 22.04 LTS → Ubuntu 24.04 LTS
  • Pythonのバージョンを更新しました。
    Python 3.10.12 → Python 3.12.3
  • データベース管理環境を更新しました。
  • 開発環境を更新しました。(元アプリサーバへ移行)

不具合の修正

  • サーバー参加直後のユーザがlevelを実行した場合に発生する不具合を修正しました。
  • ロールパネルが存在しない場合にmove_panelを実行した際に発生する可能性がある不具合を修正しました。
  • pupdateを実行した場合に発生する不具合を修正しました。
  • コマンドの実行エラーを正確に補足できない不具合を修正しました。
  • その他、いくつかの不具合を修正しました。
